To run this application:

1. Make sure your MySQL database is running
2. Open coffee-shop-backend, and change the sql credentials to yours (username and password)
3. Start the server by running “npm run start’
4. Open coffeeShop in the terminal and type “npm run start” to start the frontend
5. Follow the instructions the readme file in coffeeShop
